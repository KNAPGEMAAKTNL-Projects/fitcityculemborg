# FitCity Culemborg

Demo website for FitCity fitness club in Culemborg, Netherlands.

## Tech Stack

- Astro
- Tailwind CSS
- TypeScript

## Development

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

Deployed via Cloudflare Pages. Pushes to `main` automatically deploy to production.
